## SYNOPSIS

Searches for packages from configured sources.

## SYNTAX

```powershell
Find-ADTWinGetPackage [[-Query] <String>] [-MatchOption <String>] [-Command <String>] [-Count <UInt32>]
 [-Id <String>] [-Moniker <String>] [-Name <String>] [-Source <String>] [-Tag <String>] [<CommonParameters>]
```

## DESCRIPTION

Searches for packages from configured sources.

## EXAMPLES

### EXAMPLE 1

```powershell
Find-WinGetPackage -Id Microsoft.PowerShell
```

This example shows how to search for packages by package identifier.
By default, the command searches all configured sources.
The command performs a case-insensitive substring match against the PackageIdentifier property of the packages.
