## SYNOPSIS

Lists configured WinGet sources.

## SYNTAX

```powershell
Get-ADTWinGetSource [[-Name] <String>] [<CommonParameters>]
```

## DESCRIPTION

Lists the configured WinGet sources.

## EXAMPLES

### EXAMPLE 1

```powershell
Get-ADTWinGetSource
```

Lists all configured WinGet sources.

## PARAMETERS

### -Name
