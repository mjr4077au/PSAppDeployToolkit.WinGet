## SYNOPSIS

Repairs the installation of the WinGet client on your computer.

## SYNTAX

```powershell
Repair-ADTWinGetPackageManager [<CommonParameters>]
```

## DESCRIPTION

This command repairs the installation of the WinGet client on your computer by installing the specified version or the latest version of the client.
This command can also install the WinGet client if it is not already installed on your machine.
It ensures that the client is installed in a working state.

## EXAMPLES

### EXAMPLE 1

```powershell
Repair-ADTWinGetPackageManager
```

This example shows how to repair they WinGet client by installing the latest version and ensuring it functions properly.
